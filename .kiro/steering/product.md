# Ray Monorepo Product Overview

Ray is a modern web application suite built as a monorepo, featuring three distinct applications:

## Applications

- **Web App** (`apps/web`) - Main collaborative web application with rich text editing, real-time collaboration via Liveblocks, AI integration, and user authentication
- **Documentation** (`apps/docs`) - Documentation site built with Fumadocs for project documentation and guides  
- **Landing Page** (`apps/landing`) - Marketing website for product promotion and user acquisition

## Key Features

- **Real-time Collaboration** - Multi-user editing with Liveblocks integration
- **Rich Text Editing** - Custom Tiptap editor with BlockNote integration
- **AI Integration** - OpenAI and Google AI SDK integration for intelligent features
- **Authentication** - Better Auth system with Convex backend
- **Modern UI** - Inspired by Linear and Notion with shadcn/ui components
- **Backend as a Service** - Powered by Convex for real-time data and serverless functions

## Design Philosophy

- **Minimal & Clean** - Uncluttered interfaces with high attention to detail
- **Best-in-class DX** - Exceptional developer experience with modern tooling
- **Performance First** - Optimized builds and runtime performance
- **Accessibility** - WCAG 2.1 AA compliance throughout
